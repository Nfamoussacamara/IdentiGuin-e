![Status](https://img.shields.io/badge/status-Production--Ready-blue.svg)

**IdentiGuinée** est une solution technologique de rupture conçue pour moderniser l'état civil en République de Guinée. En s'appuyant sur la puissance de la blockchain, le projet élimine structurellement la corruption et les intermédiaires dans la délivrance des documents officiels.

---

## 🚀 Vision & Problématique

En Guinée, l'obtention d'un document d'identité est souvent synonyme de parcours du combattant :
- **Opacité** : Processus manuels favorisant les paiements informels.
- **Lenteur** : Délais d'attente imprévisibles et dossiers perdus.
- **Falsification** : Risque élevé de documents contrefaits.

**IdentiGuinée répond à ces enjeux par la "Transparence par le Code".**

---

## 💎 Piliers de la Solution

### 1. Blockchain Immuable (NaissanceChain)
Chaque document est ancré sur la blockchain **NaissanceChain**, rendant toute modification ultérieure ou falsification techniquement impossible.

### 2. Automatisation Totale (Smart Contracts)
La validation des données est gérée par des contrats intelligents. Si les conditions sont remplies, le document est généré et signé cryptographiquement sans aucune intervention humaine.

### 3. Souveraineté Numérique
Un système décentralisé qui redonne le pouvoir aux citoyens et la crédibilité aux institutions de l'État.

## 🛠️ Stack Technique

Le projet utilise les technologies les plus modernes pour garantir une expérience utilisateur fluide et une sécurité maximale :

## 💻 Installation & Développement

Pour lancer le projet localement :

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/Nfamoussacamara/IdentiGuin-e.git
   cd IdentiGuin-e
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

4. **Build pour la production**
   ```bash
   npm run build
   ```
